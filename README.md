# Sistema de Procesamiento de Marcaciones FIAS

## Descripción
Sistema web completo para procesamiento automático de marcaciones biométricas y de Microsoft Teams para la organización FIAS y sus proyectos asociados.

## Características Principales

### ✅ Procesamiento Automático
- Procesa archivos de Microsoft Teams y reloj biométrico
- Identificación automática de funcionarios por ID y nombre
- Validación de datos y detección de errores
- Unificación de datos en un solo formato

### ✅ Gestión Completa
- **Funcionarios**: CRUD completo con activación/desactivación
- **Proyectos**: FIAS, FEIG, FAP, BIOECONOMIA, CONSERVA AVES, PASNAP-INABIO, PASF
- **Configuración**: Parámetros personalizables del sistema
- **Backup**: Sistema automático de respaldo

### ✅ Reportes y Análisis
- Estadísticas por proyecto y funcionario
- Gráficos interactivos
- Exportación a Excel y CSV
- Generación de resúmenes automáticos

### ✅ Escalabilidad
- Fácil adición de nuevos proyectos
- Sistema modular y extensible
- Base de datos JSON local (puede migrarse a MySQL/MongoDB)
